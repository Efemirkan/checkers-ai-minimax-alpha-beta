# Checkers AI with Minimax and Alpha-Beta Pruning

A Python checkers game with a tkinter GUI and an AI opponent using Minimax with Alpha-Beta pruning.

## Planned Features
- Human vs AI gameplay
- Tkinter graphical interface
- Adjustable AI difficulty
- Forced capture rule
- Multi-step captures
- King promotion
- Regicide rule
- Move hints
- Suggested move feature
- Rules popup
- Instructions popup
- Game-over detection

## Technologies Used
- Python
- tkinter
- Minimax
- Alpha-Beta pruning
